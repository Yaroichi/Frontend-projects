1.Marketing Email Automation Landing Page (Design folder)

A modern, clean landing page designed to promote an email automation and customer engagement platform. The page presents the product’s key features, value proposition, and social proof to help convert visitors into demo requests.

 Features
- Responsive layout that adapts to all screen sizes (desktop and mobile)
- Hero section with primary call-to-action (“Get a demo”)
- Features section that explains core capabilities of the service
- Visual content blocks to maintain attention and support information
- Trusted by brands section to build customer credibility

 Tech Stack

- HTML5(with use of BEM technique)
- CSS3
- Flexbox & Grid layout
- Google Fonts / custom font pairing



 Design Approach

The design focuses on delivering a professional and minimalist user experience. Visual hierarchy is created through typography, spacing, and reusable UI components. Rounded shapes and soft colors help maintain a clean and approachable aesthetic.

## Project Purpose

This project demonstrates practical frontend development skills, including page structuring, responsive layout building, and visual composition. It can be used as part of a personal or commercial marketing website.

3.Online guide(guide folder)

This landing page was created as part of a web development project to present an online city guide service.  
The landing introduces the destination, available tour routes, and the value of using an online guide while traveling.

Language Requirement

The project was designed for international use, with the content specifically presented in Russian to make the information accessible to a Russian-speaking audience.

 Features

- Hero section with a strong call-to-action
- Route showcase section with image + description layout
- Section highlighting the advantages of using the online guide
- Clean visual layout with large spacing and readable typography
- Consistent color theme to support travel and exploration mood

Tech Stack

- HTML5(with use of BEM technique)
- CSS3
- Flexbox/Grid layout
- Google Fonts

 Project Purpose

This project helped to improve skills in:
- Multi-section page layout
- Working with images and visual balance
- Structuring travel-related content in a clear way
- Designing for a specific target audience and language requirement
